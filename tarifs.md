# Les différents tarifs de l'espace de coworking
Plusieurs formules existent:
- Location bureau : 480€ / mois
- ~~Location bureau partagé : 180€ / mois~~ Plus de place disponible
- Openspace : 102€ / mois
- A la carte :
    - 1j : 12€
    - demi journée : 6€

Les tarifs s'entendent avec TVA à 20% incluse.
